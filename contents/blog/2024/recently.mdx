<Date date="2023-10-20" />

To change dock icon for [calibre](https://calibre-ebook.com/) and its ebook-viewer on macOS:

1. Navigate to `Applications` directory and replace the main icon;
2. Right-click to show package content, go to `/resources/images` and replace the `library.png`;
3. Access user folder by pressing <kbd>Command</kbd> + <kbd>Shift</kbd> + <kbd>H</kbd> and go to `/Library/Preferences`. If the `/calibre/resources/images` directory doesn’t exist, create it manually. Place the `viewer.png` inside it.
